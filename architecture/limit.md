# 使用限制

### 功能限制

 * Backend节点默认3副本起。
 * 为保证集群稳定性及数据安全性，不支持节点和数据盘缩容。

### 命名限制

| 对象     | 限制                                                         |
| -------- | ------------------------------------------------------------ |
| 数据库名 | 数据库名不能以内置数据库如“system”命名。                     |
| 表名     | 表名不包含引号、感叹号（!）和空格。表名不能是SQL保留关键字。 |
| 列名     | 列名不包含引号、感叹号（!）和空格。列名不能是SQL保留关键字。 |
| 密码     | 密码长度限8-30个字符。 <br />不能包含[A-Z],[a-z],[0-9]之外的非法字符。 <br />需同时包含大小写字母和数字。 |
